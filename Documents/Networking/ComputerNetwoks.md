1. What is a Computer Network?
	- Communication between two or more network interfaces.
	- Every device on the network will have an IP address assigned to its network interface.
2. OSI Model
	- Open System Interconnection
![[OSIModel.png]]

- The basic elements of a layered model are
	- services
		- Set of actions that a layer offers to another higher one.
	- protocol
		- Set of rules that a layer uses to exchange information.
	- interfaces
		- Communication between layers
